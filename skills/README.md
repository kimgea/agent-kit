# Skills

Purpose: reusable capability modules that can be attached to one or more agents.

Read `GODOT_ROUTING.md` before Godot implementation work.
Do not load every Godot skill by default. Start with the recommended core set and add specialist skills only when the task or project needs them.

Place here:
- narrowly scoped operational instructions
- supporting references needed by that skill
- skill-local validation when useful

Do not place here:
- complete agent personas
- provider-specific wrappers
- illustrative placeholder skills

Real asset path:

```text
skills/<skill-id>/
  manifest.yaml
  ...
```
