# Agent Instructions

This vault is a shared human and agent workspace for `{{project_name}}`.

## Read First

Before substantial work, read:

1. `00 Start Here/Start Here.md`
2. `01 Coordination/Current Focus.md`

## Core Rules

- Treat this vault as shared memory, not a scratchpad.
- Put raw imported material into `05 Intake/`.
- Move processed raw batches to `65 Sources/` instead of leaving them in intake.
- Put brainstorming into `10 Ideas/` or `20 Explorations/`.
- Put durable outputs into `30 Designs/`, `40 Requirements/`, `50 Decisions/`, or `60 Reference/`.
- Do not treat `05 Intake/` or coordination notes as canonical truth.
- When promoting information, link the source note instead of deleting history.
- Mark obsolete or rejected work clearly instead of overwriting it.

## Writeback

After substantial work:

1. Update or create the durable note that holds the result.
2. Leave a concise handoff in `01 Coordination/Handoff Log.md` if the work matters to others.
3. Update `01 Coordination/Current Focus.md` when priorities or blockers change.
