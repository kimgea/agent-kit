# Workflow Rules

## Promotion Flow

Use this default flow:

1. raw material goes into `05 Intake/To Process`
2. early thoughts go into `10 Ideas/`
3. alternatives and analysis go into `20 Explorations/`
4. chosen direction becomes a note in `30 Designs/`
5. build-facing constraints become notes in `40 Requirements/`
6. major decisions become notes in `50 Decisions/`

## Intake Processing Rule

When working from existing material:

1. keep the source in `05 Intake/` while it is being processed
2. create or update the proper durable note
3. add links between source and output
4. move processed raw source material to `65 Sources/`

## Coordination Rule

`01 Coordination/` is for active collaboration, not the final home of project knowledge.
