# Prompts

Purpose: reusable prompt modules and recipes.

Suggested internal organization for real assets:
- composable modules
- higher-level recipes

Place here:
- style modules
- output format instructions
- reusable task patterns

Do not place here:
- full agents unless the prompt is the asset itself
- duplicated provider formatting unless necessary

Real asset path:

```text
prompts/<prompt-id>/
  manifest.yaml
  ...
```
